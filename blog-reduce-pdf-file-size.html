How to Reduce PDF File Size — 7 Practical Tips That Actually Work

Published: April 18, 2026  ·  5 min read  ·  GPTPayer Team

We have all been there — you try to attach a PDF to an email and it is rejected because the file is too large. Or an online portal only accepts files under 2MB and your PDF is 15MB. This is an incredibly common problem. Here are 7 practical tips to dramatically reduce your PDF file size.

1. Reduce Image Resolution Before Converting

Images embedded in PDFs are the single biggest contributor to file size. If your images are saved at 300 DPI or higher, that is overkill for screen viewing — 150 DPI is perfectly sharp on any monitor, and 96 DPI is often sufficient. When saving your JPG images, set the quality to 70–80%. The difference is virtually invisible to the human eye, but the file size can drop by 50% or more.

2. Use the Correct Page Size

If you accidentally select A3 when you actually need A4, your PDF will be unnecessarily large. Always double-check the page size setting before converting. It sounds simple, but it is one of the most commonly overlooked causes of bloated PDFs.

💡 Quick Check: Open your PDF in a browser or PDF viewer and check the document properties. If the page size is wrong, reconvert with the correct settings.

3. Use JPG Instead of PNG for Photographs

PNG is a lossless format — perfect quality, but much larger file sizes. For photographs, JPG is significantly more efficient. A photo saved as PNG might be 8MB; the same photo saved as JPG at 80% quality might be 800KB — with virtually no visible difference. Use PNG only when you need a transparent background or when the image contains sharp text or logos.

4. Remove Unnecessary Pages

Blank pages, duplicate pages, or pages that are no longer relevant all add to the file size. Review your PDF before sharing and remove anything that does not need to be there. Every page you eliminate reduces the file size proportionally.

5. Do Not Embed Fonts (When Possible)

When exporting PDFs from Word or PowerPoint, look for an option like "Do not embed fonts." Standard system fonts such as Arial, Times New Roman, and Calibri are available on virtually every computer — they do not need to be embedded in the file. Skipping font embedding can reduce file size by 10–30%.

6. Use an Online PDF Compression Tool

Several free tools are specifically designed to compress PDFs — Smallpdf, ilovePDF, and PDF2Go are popular options. These tools recompress images and remove unnecessary metadata. A 10MB PDF can often be brought down to 1–2MB with minimal quality loss.

7. Set the Right Resolution When Scanning

If you are scanning physical documents, set your scanner to 150–200 DPI rather than 600 DPI. For documents that are not in color, use Grayscale or Black & White mode instead of full color. These two changes alone can reduce a scanned PDF's size by 5–10 times.

Real-World Example

Suppose you have 5 photos to include in a PDF. At their original size, each JPG is 3MB — that is 15MB total. If you first resize and resave them at 150 DPI and 80% quality, each file drops to around 400–500KB. Your final PDF would be around 2–3MB — an 80% reduction in size with no perceptible quality loss.
